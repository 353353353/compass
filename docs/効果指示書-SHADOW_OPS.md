## この動画の"背骨"（山場マップ）

離脱しやすいポイントを、山で挟んで持たせる設計になってる：

```
フック(S1) → 完成カタルシス(S4)① → うめぼしコメディ(S5) →
早口&ボケ(S7) → 影分身アタック(S8) → 派手なウニ回収(S9)② → CTA(S10)
```

**約1〜1.5分に1回、山（笑い or 驚き or カタルシス）が来る**リズム。これが視聴維持の背骨。
次回作もこの間隔を意識すると崩れない。
